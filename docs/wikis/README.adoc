== Core concepts

Panoply is three artefacts — a *checked Core*, *explicit evidence*, and a
*safety-envelope manifest* — governed by one rule: _no guarantee without an
envelope; no envelope without evidence; no composition without a composition
rule_. The xref:GLOSSARY.adoc[Glossary] defines each term; the
xref:../architecture/DESIGN-DISCIPLINE.adoc[charter] is the source of truth.
